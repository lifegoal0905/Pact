pub extern fn create_mock_server(pact_str: *const c_char) -> int32_t

External interface to create a mock server. A pointer to the pact JSON as a C string is passed in, and the port of the mock server is returned.

Errors

Errors are returned as negative values.

Error Description
-1 A null pointer was received
-2 The pact JSON could not be parsed
